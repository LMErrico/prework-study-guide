HTML

  • HTML attributes defines additional information about an element and are always defined in the beginning element.
  • Semantic HTML elements indicate the role of their content, as well as to give structure to the content of the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The section elements define a section in a document that contains thematic content.
  • An element refers to the entire element from opening tag to closing tag.
  • A tag refers only to what appears inside the angle brackets.
  • The element that holds all the child elements is known as the parent element.
  • The elements that are nested inside other elements are called child elements.

Git

  • Git is a version control system that lets you manage and keep track of each version of your proyect.
  • To access Git, we use the command line.
  • Git commands always start with git.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: add a file as it looks now to your next commit.
  • giy pull origin main: pull to get the latest version.
  • Git is version control software and GitHub is a cloud-based service that hosts our repos.
